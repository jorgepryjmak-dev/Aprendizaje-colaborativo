Etiquetas Semánticas Estructurales

Es una etiqueta contenedora que no tiene semánticas
header - Cabecera de un sitio web o de una seccion
main - Define la seccion principal del documento, solo puede existir una etiqueta main por documento
footer - Pie de pagina de un sitio web o de una sección
nav - Representa la navegación
article - Representa una seccion autocontenido (que por si sola se explica)
aside - Representacontenido complementario o secundario(publicidad)
section - Representa una seccion de contenido genérico
address - Representa una dirección física

Etiquetas de Bloque VS Etiquetas de Linea

La etiqueta por excelencia es la div, una etiqueta de bloque es aquellas que genera saltos de linea con sus etiquetas hermanas
La etiqueta de linea por excelencia es el span, una etiqueta de linea solo ocupa el espacio necesario que continene su contenidohola soy otra span
